LESSON 12: Bitwise XOR - The Magic Instruction
════════════════════════════════════════

XOR OPERATION

  xorq SOURCE, DEST   →   DEST = DEST ^ SOURCE

XOR (eXclusive OR) produces 1 if bits are DIFFERENT:

  1 XOR 1 = 0
  1 XOR 0 = 1
  0 XOR 1 = 1
  0 XOR 0 = 0


THE MAGIC PROPERTY

  X XOR X = 0

Anything XOR'd with itself is zero!

  xorq %rax, %rax     # RAX = 0

This is the fastest, most efficient way to zero a register.


WHY NOT MOV?

  movq $0, %rax       # 7 bytes, needs immediate value
  xorq %rax, %rax     # 3 bytes, CPU recognizes this pattern

The CPU has special optimizations for "xor reg, reg" - it knows you're
zeroing the register and doesn't even need to wait for the old value.


OTHER XOR TRICKS

  - Swap without temp: a ^= b; b ^= a; a ^= b;
  - Toggle specific bits: value ^= mask
  - Simple encryption: data ^= key (XOR again to decrypt)
