LESSON 05: The ADD Instruction
════════════════════════════════════════

ADDITION IN ASSEMBLY

  addq SOURCE, DEST

This computes:  DEST = DEST + SOURCE

The result REPLACES the destination. The source is unchanged.


EXAMPLES

  addq $10, %rax     # rax = rax + 10
  addq %rbx, %rax    # rax = rax + rbx
  addq $1, %rdi      # rdi = rdi + 1 (increment)


FLAGS

Addition also sets CPU flags:
  - ZF (Zero Flag) = 1 if result is zero
  - SF (Sign Flag) = 1 if result is negative
  - CF (Carry Flag) = 1 if unsigned overflow
  - OF (Overflow Flag) = 1 if signed overflow

We'll use these later for conditional jumps.


THE PROBLEM

You have 10 in %rdi.
You need 42 in %rdi.

The difference is: 42 - 10 = ???

You need to add that difference.


YOUR TASK

Write an addq instruction that adds the right value to %rdi.

The general form is:
  addq $VALUE, %REGISTER

What should VALUE be to turn 10 into 42?
