Exercise 21: Stack Frames

A stack frame is the region of stack used by a function.
The standard prologue/epilogue:

PROLOGUE (at function start):
  pushq %rbp          # save old base pointer
  movq %rsp, %rbp     # set new base pointer

EPILOGUE (before return):
  popq %rbp           # restore old base pointer
  ret

%rbp (base pointer) marks the bottom of our frame.
This creates a stable reference point for local variables.

Complete the prologue and epilogue in my_func.
