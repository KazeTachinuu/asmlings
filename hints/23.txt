WHAT IS A LOOP?

A loop is just:
  1. Do some work
  2. Check a condition
  3. Jump back if condition is true


THE PATTERN

  movq $N, %rcx           # Initialize counter

loop_start:
  # ... do work ...

  decq %rcx               # counter--
  jnz loop_start          # if counter != 0, repeat


DECQ

  decq %rcx

Decrements %rcx by 1 AND sets flags.
If the result is zero, ZF = 1.


JNZ

  jnz label

"Jump if Not Zero" - jumps if ZF = 0.

Together: decq + jnz loops until counter reaches 0.


EQUIVALENT C

  for (int i = N; i > 0; i--) {
      // do work
  }


THE CODE

You have:
  - Result accumulator: %rdi (starts at 0)
  - Loop counter: %rcx (starts at 5)
  - Loop body: addq $1, %rdi

You need to add:
  - Decrement the counter
  - Jump back if not zero


YOUR TASK

After addq $1, %rdi, add two instructions:
  1. Decrement %rcx
  2. Jump to loop_start if %rcx is not zero
