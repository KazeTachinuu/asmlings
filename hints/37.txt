To call a C function like puts:

1. Set up the function properly:
   main:
       pushq %rbp
       movq %rsp, %rbp

2. Load the string address into %rdi:
   leaq message(%rip), %rdi

3. Call the function:
   call puts

4. Return 0:
   xorq %rax, %rax
   popq %rbp
   ret

Note: puts adds a newline automatically!
