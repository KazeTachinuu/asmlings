greet - Print a Greeting
════════════════════════════════════════

THE SETUP

The exercise already provides:
  - "Hello, " string in .rodata with its length
  - argv[1] is loaded into %r12
  - strlen function is defined

You need to:
  1. Print "Hello, "
  2. Get the length of argv[1]
  3. Print argv[1]


STEP 1: PRINT "Hello, "

Use the write syscall:
  movq $1, %rax           # syscall: write
  movq $1, %rdi           # fd: stdout
  leaq hello(%rip), %rsi  # buffer: the "Hello, " string
  movq $hello_len, %rdx   # length: use the constant
  syscall


STEP 2: GET LENGTH OF argv[1]

Before calling strlen, put the string pointer in %rdi:
  movq %r12, %rdi         # pointer to argv[1]
  call strlen             # returns length in %rax

But wait: %r12 will be needed again for the write!
And %rax will be overwritten by the syscall number.

Save both values before setting up write.


STEP 3: PRINT argv[1]

Set up write using your saved values.
